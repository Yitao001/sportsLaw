# WADA禁用清单与检测程序完全指南

> WADA（世界反兴奋剂机构）制定的《禁用清单》每年1月1日更新，适用于所有签署《世界反兴奋剂条例》的运动员。

---

## 一、禁用清单（Prohibited List）结构

### S 类：任何时间禁用物质
| 类别 | 典型物质 | 常见用途/误用 |
|-------|-----------|----------------|
| S1 合成代谢类固醇 | 睾酮、诺龙 | 增肌 |
| S2 肽类激素 | EPO、HGH | 增红细胞/恢复 |
| S3 β2-受体激动剂 | 沙丁胺醇（限量内允许） | 哮喘治疗 |
| S4 激素与代谢调节剂 | 他莫昔芬 | 调节雌激素 |
| S5 利尿剂与其他掩蔽剂 | 呋塞米 | 稀释尿液 |

### 赛内禁用物质（S6–S7）
- **S6 刺激剂**：安非他明、可卡因、麻黄碱（部分）
- **S7 麻醉剂**：吗啡、海洛因（严格禁用）
- **S8 大麻（ cannabinoids）**：THC 代谢物超标

### 特定运动禁用
- **β-受体阻断剂**（射击、射箭、汽车运动等需要手部稳定的项目）

---

## 二、检测程序

### 赛内检测（In-Competition）
1. 检测官（DCO）在赛后 1 小时内通知运动员
2. 运动员选择排尿地点（有隐私屏障）
3. 提供 A 瓶和 B 瓶各 90ml 尿液（或血液）
4. 运动员有权要求在场陪同人员
5. **拒绝检测 = 阳性（ADRV）**

### 赛外检测（Out-of-Competition）
1. 运动员须通过 ADAMS 系统申报行踪
2. 检测官可随时上门检测（需出示证件）
3. 一年内 **3次"未能完成检测"**（Whereabouts Failure）= ADRV
4. 申报行踪截止时间：每季度末前（具体日期 WADA 公告）

### A 瓶阳性后程序
1. 运动员收到阳性通知（通常 2–4 周内）
2. 可申请 B 瓶复检（须在规定时间内提出，通常 10 天）
3. 同时收到完整的实验室文件包（LDP）
4. 可向国家反兴奋剂机构提出申诉

## 三、ADRV 处罚标准（WADA Code 2021 版）

| 违规情形 | 禁赛期 |
|---------|---------|
| 首次故意使用 | **4年** |
| 首次非故意 | **2年** |
| 特定物质（非故意） | 最高 2 年（可降至警告） |
| 二次违规 | 禁赛期加倍（最高终身） |
| 持用禁用物质 | 4年（运动员辅助人员） |
| 贩卖/运输 | 4年至终身 |
| 向未成年人提供 | 终身禁赛 |

### 减轻处罚情节
- **无过错/无重大过错**：免除或大幅减少禁赛（需举证）
- **实质性协助**：举报他人→ 可减少 25%–75%
- **及时认罪**：在证据确凿前主动承认 → 可减刑

## 四、治疗用药豁免（TUE）

### 申请条件
运动员因医疗需要必须使用禁用物质时，可提前申请 TUE。

### 申请流程
1. 由执业医师出具医疗诊断证明
2. 填写 TUE 申请表（WADA 标准表格）
3. 提交至所属国际联合会 TUE 委员会
4. **提前 30 天申请**（紧急情况可事后 24 小时内补办）

### 常见获批 TUE 情形
| 物质 | 医疗用途 |
|-------|---------|
| 沙丁胺醇 | 哮喘 |
| 皮质类固醇（局部） | 皮肤/关节炎症 |
| 胰岛素 | 1 型糖尿病 |

> ⚠️ **未获批 TUE 而使用 = ADRV**

---

## 五、运动员权利

- 知情权：完整了解指控和所有证据
- 听证权：在独立机构前陈述和辩护
- 申诉权：对一审不服可逐级申诉至 CAS
- 隐私权：检测信息受法律保护
- 申请 TUE 复核权

---

## 六、重要注意事项

1. **严格责任原则**：无论是否故意，检出即构成违规
2. **补剂风险自负**："被污染的补剂"不是免责理由（除非能证明无过错）
3. **出行前检查**：比赛目的地国家可能视某些药物为禁用
4. **保存所有医疗记录**：TUE 申请和申诉时的重要证据
