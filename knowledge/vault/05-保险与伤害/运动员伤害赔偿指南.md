# 运动员伤害赔偿指南

## 一、伤害责任划分

### 雇主责任（俱乐部）
| 情形 | 俱乐部责任 |
|------|-----------|
| 训练中受伤（正常训练） | 工伤，承担医疗费 + 工资 |
| 提供不安全训练条件 | 承担全部赔偿责任 |
| 拒绝支付医疗费 | 运动员可申诉至劳动仲裁机构 |

### 第三方责任
- 对方运动员**故意或重大过失**导致的伤害 → 可提起民事诉讼索赔
- 赛事组织者未尽安全保障义务 → 可索赔
- 器材/场地缺陷 → 可向生产商/场地管理者索赔

### 运动员自身过失
- 普通过失（比赛正常风险） → 自担后果
- **重大过失或故意伤害** → 可能被追偿

## 二、职业终结保险（Career-Ending Insurance）

- 保障因伤/病永久丧失运动能力的运动员
- 通常由俱乐部或联赛购买（如英超、NBA）
- 赔付金额：通常为合同剩余价值的 50%–100%
- **审查要点**：投保时须如实告知既往伤病史

## 三、强制保险

| 联赛 | 强制保险要求 |
|------|-------------|
| 英超 | 俱乐部须购买意外伤害险（最低 100 万英镑） |
| NBA | 联盟统一购买职业健康与残疾保险 |
| 中超 | 中国足协要求俱乐部购买工伤保险 |

## 四、伤害豁免条款审查

球员合同中须注意：
- 豁免范围：仅限"普通过失"，**重大过失或故意**不可豁免
- 重伤定义：须明确何种程度构成"职业终结"
- 复出条款：受伤后复出条件须书面约定

> ⚠️ NFL 脑震荡集体诉讼（10 亿美元和解）是职业伤病索赔的里程碑案例
