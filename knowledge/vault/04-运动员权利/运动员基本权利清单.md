# 运动员基本权利清单

> 本清单汇总各国际体育组织通用运动员权利，适用于所有项目运动员。

---

## 一、程序性权利

### 正当程序权（Due Process）
在任何纪律程序（禁赛、罚款、取消资格等）中，运动员有权：
1. **事前告知**：在决定前收到指控通知和证据
2. **申辩权**：在决定前提交书面申辩意见
3. **听证权**：要求举行听证会（部分机构可拒绝，但须书面说明理由）
4. **上诉权**：对不利决定向上级机构或 CAS 上诉

### 翻译权
- 纪律程序中使用本国语言的权利
- 如机构使用外语，运动员有权要求提供翻译

---

## 二、隐私权

### 反兴奋剂检测中的隐私保护
- 采样人员须出示正式证件
- 采样过程须保护运动员隐私（独立采样隔间）
- 检测结果信息仅限授权人员知晓
- 运动员有权要求销毁超期保存的检测样本

### 行踪信息保护
- ADAMS 系统中的行踪信息仅用于反兴奋剂目的
- 运动员有权要求查阅和更正个人数据（GDPR 权利）

---

## 三、劳动权

### 运动员的法律身份
| 国家/地区 | 运动员身份认定 |
|-----------|---------------|
| 欧盟国家 | 通常认定为**雇员**（享有劳动法保护） |
| 美国 | 大学生运动员 = 学生；职业运动员 = 独立承包人 |
| 中国 | 体制内运动员 = 体制管理；职业运动员 = 合同关系 |

### 核心劳动权利
1. **获得报酬权**：按合同约定获取工资和奖金
2. **安全训练权**：拒绝不安全的训练和比赛条件
3. **不受歧视权**：不因性别、种族、宗教、性取向受不公平对待
4. **组织工会权**（部分国家）：集体谈判提高待遇

---

## 四、肖像权与商业权利

### 肖像权（Image Rights）
- 运动员对自身形象、姓名、昵称拥有知识产权
- 赞助合同中的**独家授权范围**须明确约定，避免过度让渡
- 未经授权使用运动员肖像 → 侵权，可索赔（最高可达实际损失 3 倍）

### 道德条款（Morality Clause）
- 赞助合同中常见条款：运动员发生负面新闻时，赞助商可解约
- **审查要点**：定义"负面新闻"的标准须明确，避免过于宽泛

---

## 五、平等参与权

- 禁止基于性别、种族、残疾、宗教、性取向的歧视
- CAS 多次裁定：技术上的性别检测（如雄激素规则）须符合比例原则
- 残奥运动员享有与奥运会运动员同等的法律保护框架

---

## 六、求助渠道

| 争议类型 | 求助机构 |
|---------|---------|
| 反兴奋剂处罚 | 国家反兴奋剂机构 → 国际联合会 → CAS |
| 转会/合同纠纷 | FIFA DRC → CAS |
| 歧视/骚扰 | 所属国际联合会道德委员会 → CAS |
| 性骚扰/虐待 | 国际奥委会《安全运动框架》举报机制 |
