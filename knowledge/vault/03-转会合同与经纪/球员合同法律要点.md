# 球员合同法律要点

## 合同基本要素（FIFA RSTP 要求）

| 要素 | 要求 |
|------|------|
| 最短期限 | 1 年 |
| 最长期限 | 5 年 |
| 书面形式 | 必须（口头合同无效） |
| 球员年龄 | ≥18 岁（国际转会例外见未成年人保护规则） |
| 注册窗口 | 须在转会窗口内注册生效 |

## 关键条款审查清单

- [ ] 工资支付时间和方式（逾期违约责任）
- [ ] 奖金条款（出场次数、成绩目标、具体计算方式）
- [ ] 合同终止条款（正当理由定义）
- [ ] 违约金数额和支付方式
- [ ] 形象权与赞助权划分
- [ ] 争议解决管辖（FIFA DRC / CAS / 法院）
- [ ] 道德条款（不当行为可导致解约）

## 常见纠纷类型

1. **工资拖欠**：超过 2 个月可构成正当理由终止
2. **单方面降薪**：须球员书面同意，否则构成违约
3. **侵犯形象权**：俱乐部未经授权使用球员肖像
4. **不当终止**：无正当理由解约须支付违约金
