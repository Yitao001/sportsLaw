# CAS仲裁完整指南

> CAS（体育仲裁法院）是体育领域最高仲裁机构，裁决具有终局性，可在155个国家强制执行。

---

## 一、CAS 基本概况

| 项目 | 内容 |
|------|------|
| 成立时间 | 1984年（国际奥委会设立） |
| 总部 | 瑞士洛桑 |
| 适用法律 | 瑞士私法 |
| 裁决执行 | 《纽约公约》155个国家 |
| 官方语言 | 英语、法语（可协议其他语言） |

---

## 二、CAS 仲裁程序类型

### 1. 普通仲裁（Ordinary Arbitration）
- **适用**：合同争议、侵权纠纷、体育组织内部争议
- **时限**：申请仲裁需在争议发生后按协议约定，无约定则适用各国时效
- **费用**：仲裁员费用 5,000–30,000 瑞士法郎 + 申请费 1,000 瑞士法郎
- **时长**：通常 6–12 个月

### 2. 上诉仲裁（Appeal Arbitration）
- **适用**：对国际体育联合会、国家奥委会决定不服
- **时限**：**21天**（自收到决定书起算）
- **费用**：申请费 1,000 瑞士法郎
- **时长**：通常 3–6 个月

### 3. 临时仲裁（Ad Hoc）
- **适用**：奥运会、亚运会等大型赛事期间的紧急争议
- **时限**：**24小时内**出裁决
- **费用**：由 CAS 酌情减免
- **特点**：仅处理影响参赛资格的紧急事项

### 4. 调解（Mediation）
- **适用**：双方自愿，寻求和解
- **成功率**：约 80%
- **费用**：低于仲裁
- **结果**：调解协议可申请 CAS 确认后强制执行

## 三、上诉仲裁完整流程

### 第一步：确认上诉资格（时限内）
- 须在收到决定书后 **21天**内提交
- 确认案件属于 CAS 管辖（查看原始规则中是否约定 CAS 上诉）
- 已穷尽内部申诉程序（合同或规则有约定的）

### 第二步：准备上诉状（Memo）
须包含：
1. 当事人基本信息
2. 被上诉决定原文或认证副本
3. 事实陈述（发生了什么）
4. 法律依据（引用具体规则条文）
5. 救济请求（具体要求 CAS 如何裁决）
6. 仲裁员提名（如规则允许）

### 第三步：提交与缴费
- 提交方式：邮件至 `arbitration@tas-cas.org` 或邮寄至洛桑总部
- 份数：一式五份 + 电子版
- 缴费：1,000 瑞士法郎（经济困难可申请减免）

### 第四步：仲裁庭组成
- CAS 秘书处指定首席仲裁员
- 双方各选一名仲裁员（如规则允许）
- 仲裁员须独立、无利益冲突

### 第五步：书面程序
| 文件 | 时限 |
|------|------|
| 上诉理由书（Appeal Brief） | 上诉状提交后 20 天 |
| 答辩书（Answering Brief） | 收到上诉理由书后 20 天 |
| 反驳书（Reply Brief） | 收到答辩书后 15 天 |

### 第六步：听证会（可选但推荐）
- 口头陈述、证人交叉询问、法律辩论
- 可在线举行（CAS Practice Note 2023/1）
- 当事人有权委托律师（须在 CAS 律师名单上）

### 第七步：裁决
- 普通程序：3–6 个月
- 裁决书送达双方当事人
- 裁决为终局性

## 四、费用估算与法律援助

| 项目 | 费用范围（瑞士法郎） |
|------|----------------------|
| 申请费 | 1,000（固定） |
| 仲裁员费用 | 5,000–30,000 |
| 律师费 | 10,000–100,000+ |
| 翻译费 | 按实际发生 |
| 差旅费 | 当事人自理 |

**法律援助**：经济困难运动员可向 CAS 申请费用减免（需在申请时一并提出）

---

## 五、重要注意事项

1. **时效是硬性的**：错过 21 天通常无法补救
2. **仲裁地默认为洛桑**：除非双方书面协议其他地点
3. **裁决可在 155 国执行**：依据《纽约公约》
4. **瑞士联邦法院复核**：仅在极少数情况下（违反公共政策、仲裁员腐败等）
5. **建议聘请专业律师**：CAS 仲裁高度专业化，自行代理风险较大
