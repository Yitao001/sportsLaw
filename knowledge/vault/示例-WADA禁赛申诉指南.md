# WADA禁赛申诉完全指南

本指南帮助被禁赛的运动员了解申诉权利和完整流程。

## 你有哪些权利

作为运动员，当你收到反兴奋剂违规（ADRV）通知时，你拥有以下权利：

- **知情权**：完整了解指控内容和所有证据
- **听证权**：在独立机构前陈述辩护
- **申诉权**：对一审裁决不服可逐级申诉直至 CAS
- **申请 TUE 复核权**：如涉及治疗用药豁免争议
- **隐私权**：兴奋剂检测信息受保护

## 申诉流程图

```
阳性检测结果
    ↓
当事人收到通知
    ↓
国家/地区反兴奋剂组织（ADO）举行听证
    ↓
一审裁决（支持/减轻/撤销）
    ↓
不服 → 向国际体育联合会（IF）申诉
    ↓
IF 裁决
    ↓
不服 → 向 CAS 提起最终上诉
    ↓（或）
不服 → 向 WADA 上诉（IF 未充分制裁时）
```

## 常见申诉理由

### 1. 程序违规

以下程序违规可构成上诉理由：

- 检测样本未按规定保存（冷链断裂）
- 实验室操作不规范（质控超标）
- 当事人未被充分告知权利
- 听证程序不独立或不公正

### 2. 证据不足

- 检测实验室未排除假阳性可能
- 非典型发现（ATFs）未充分调查
- 证据链不完整

### 3. 责任减轻

争取减轻处罚的常见理由：

- **无过错**（食品污染、医疗误诊）
- **无重大过错**（清洁补剂污染、非故意摄入）
- **实质性协助**：举报他人违规可减少 25–75% 禁赛期
- **及时认罪**：在证据确凿前主动承认可获减刑

## 处罚标准速查

| 违规类型 | 默认禁赛期 |
|---------|----------|
| 首次故意使用兴奋剂 | **4年** |
| 首次非故意使用 | **2年** |
| 特定物质（非故意） | **最高2年**（可降至警告） |
| 持有禁用物质 | **4年**（运动员支持人员） |
|  Trafficking（贩卖） | **4年至终身** |
| 伪造证据 | **4年** |
| 第二次违规 | 禁赛期加倍 |

## 治疗用药豁免（TUE）争议

如果你的案件涉及 TUE：

1. 确认你的 TUE 是否在规定时间内申请（至少提前 30 天）
2. 复核 TUE 申请材料是否完整
3. 如 TUE 被拒绝，你仍可向国际 TUE 委员会（ITEC）申诉
4. 紧急情况下可先用药后申请（事后 24 小时内）

## 费用与时限

| 项目 | 要求 |
|------|------|
| 首次听证申请 | 通常免费 |
| 上诉至 IF | 参照各 IF 规则 |
| 上诉至 CAS | 申请费 1,000 瑞士法郎 |
| CAS 裁决复核 | 向瑞士联邦法院（仅限法律问题） |

**注意**：经济困难运动员可申请费用减免和公益法律援助。

## 实用建议

1. **不要拖延**：每个期限都是硬性的
2. **保留所有通信记录**：检测通知、听证通知、裁决书等
3. **寻求专业帮助**：聘请有体育法经验的律师
4. **收集有利于你的证据**：处方单、购药记录、食品来源证明
5. **考虑和解**：在证据充分的情况下与 ADO 协商减刑可能是更务实的选择
