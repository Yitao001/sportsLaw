# SportsLax — 体育法律咨询智能体

> 专注于国际体育法律的 AI 智能顾问，基于 RAG + LLM 提供专业法律咨询服务。

---

## 核心功能

| 功能 | 端点 | 说明 |
|------|------|------|
| 🗣️ 智能法律对话 | `POST /chat` | RAG + LLM 回答体育法律问题，支持多轮对话与流式输出 |
| 📋 律师信息清单报告 | `POST /report` | 运动员描述情况后，自动生成可交给律师的结构化报告 |
| 🏅 体育运动百科 | `POST /sport` | 查询运动项目的规则、管理机构及法律特点 |
| 📚 运动列表 | `GET /sports/list` | 获取已收录的所有运动项目 |
| 🔄 知识库同步 | `POST /knowledge/reload` | 手动重建向量库，同步 Obsidian vault 文档 |
| 📂 Vault 状态 | `GET /knowledge/vault` | 查看 vault 文档列表 |
| 🔍 健康检查 | `GET /health` | 检查服务与 LLM 状态 |

---

## 覆盖的法律领域

- **CAS 仲裁**：程序、时效、裁决执行
- **WADA 反兴奋剂**：禁用清单、TUE豁免、处罚与申诉
- **FIFA 转会规则**：DRC争议解决、培训赔偿、团结机制
- **运动员资格与国籍**：代表国规则、等待期、归化
- **运动员权利保护**：正当程序、隐私权、劳动权
- **体育合同与赞助**：道德条款、排他性、解约风险
- **反歧视与平等**：性别、种族、残障平等原则
- **媒体转播权**：版权、简短报道权、OTT许可
- **运动员伤害与保险**：工伤赔偿、职业终结保险

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

复制 `.env.example` 并填入配置：

```bash
cp .env.example .env
```

**.env 关键配置项：**

```env
# 模型提供商（tongyi / openai / siliconflow / anthropic）
MODEL_PROVIDER=tongyi
CHAT_MODEL_NAME=qwen-max
EMBEDDING_MODEL_NAME=text-embedding-v4

# 通义千问（阿里云）
DASHSCOPE_API_KEY=your_key_here

# 或 OpenAI 兼容接口
# OPENAI_API_KEY=sk-xxx
# OPENAI_BASE_URL=https://api.openai.com/v1

# 安全配置（留空则为开发模式，无需 API Key）
API_KEY=
CORS_ORIGINS=*
RATE_LIMIT_PER_MINUTE=60
```

### 3. 启动服务

```bash
python api.py
# 或
uvicorn api:app --reload --port 8000
```

访问 Swagger UI：http://localhost:8000/docs

## API 使用示例

### 功能一：智能法律对话

```bash
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{
    "question": "运动员被检测出兴奋剂阳性后有哪些申诉途径？",

  }'
```

### 功能三：体育运动百科查询

```bash
curl -X POST http://localhost:8000/sport \
  -H "Content-Type: application/json" \
  -d '{
    "sport_name": "足球",
    "question": "FIFA转会规则有哪些法律风险？"
  }'

```

   │
   ▼

## 扩展知识库

### 方式一：Obsidian 外挂知识库（推荐）

将 `knowledge/vault/` 目录用 Obsidian 打开，即可像编辑普通笔记一样扩充法律知识：

```
knowledge/vault/
├── 00-索引与导航/             # 知识库总索引
├── 01-仲裁程序/               # CAS 仲裁完整指南、仲裁员指定规则
├── 02-反兴奋剂/               # WADA 禁用清单与检测程序、TUE 指南
├── 03-转会合同与经纪/          # FIFA 转会规则、球员合同要点、经纪人制度
├── 04-运动员权利/             # 运动员基本权利清单
├── 05-保险与伤害/             # 运动员伤害赔偿指南
├── 06-未成年人保护/            # 未成年运动员保护规则
├── 07-比赛操纵与赌博/          # 比赛操纵法律红线
├── 08-媒体版权与转播权/        # 运动员肖像权与媒体权指南
├── 09-退役与社会保障/          # 运动员退役保障完全指南
├── 10-反歧视与平等/            # 体育反歧视法律框架
├── 11-案例库/                 # CAS 经典案例汇编
├── 12-各运动专项/             # 各运动法律特点对比
└── .templates/                # 笔记模板
```

Obsidian .md 文件格式（支持 frontmatter 元数据）：

```markdown
---
title: 文档标题
category: 分类
tags: [tag1, tag2]
---

# 正文标题

正文内容...
```

**同步方式**（三选一）：
- **自动监听**（默认）：保存 .md 后自动同步（需安装 `watchdog`）
- **手动接口**：调用 `POST /knowledge/reload`
- **重启服务**：服务启动时自动重建

**查看 vault 状态**：`GET /knowledge/vault`

### 方式二：代码追加

```python
from langchain_core.documents import Document
from rag.rag_engine import add_custom_documents

docs = [
    Document(
        page_content="你的自定义法律文本内容...",
        metadata={"source": "来源名称", "category": "分类"}
    )
]
add_custom_documents(docs)
```
